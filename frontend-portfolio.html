Intense2

Intense2 is multipurpose website template built with Pug, Sass, and JavaScript. It includes:

  • 150+ pages;
  • 120+ reusable components;
  • 40+ child templates based on parent template.

The main goal when creating this template was to achieve maximum modularity and independence of components. This allowed us to build many other websites and pages based on Intense2.

The good quality of this website template allowed to sell it more than 4,000 times.

My contribution:

  • Management of team members.
  • Developing JavaScript plugins and integrating existing ones.
  • Creating visual components by using Pug and Sass.
  • Creating pages markup.
  • Creating child website templates.

Used tools & Technologies:

JavaScript, HTML, CSS, Pug, Sass, Gulp, Node.js, Express.js, jQuery, Bootstrap, PHP, PhpStorm, Figma.

MonsterAdmin Bundle

Admindex, Adminifix, Emerald, IoDashboard, and Concrete are parts of MonsterAdmin Bundle. The main goal in developing the admin dashboards was the same as in Intense2, to achieve maximum modularity and independence of components. Each of this admin dashboards includes:

  • 50+ pages;
  • 80+ components;
  • documentation as part of admin dashboard template;
